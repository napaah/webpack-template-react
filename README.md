# ract-template 

## 所需环境及安装依赖注意事项
本项目，是webpack-react 模板，构建与2022年中旬
建议使用 `yarn` 包管理器 `npm install -g yarn` 
+ Node.js 12+
+ npm 6+
+ webpack 5.73.0
+ webpack-cli 4.10.0
+ react 18.2.0、readt-dom 6.3.0  支持 `Hooks` 特性


### 依赖安装
```bash
yarn 或者 npm install
```

## package.json scripts
+ prepare 安装husky
+ start 本地开发命令 设置node占用内存为8G
+ dev 本地开发命令
